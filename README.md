## About The App/ How to use:

This is a simple interactive app that will let you access NASA's photograph of the day! Select the date you would like from the date input and click the 'fire!' button, 
it will show you the photopgraph with the explanation.


## How It's Made:

Created using HTML, CSS, JS and Font Awesome. Application utilizes NASA API to be able to retrieve daily image of the day and description.

## Tech Used:
HTML, CSS, JavaScript



## Lessons Learned:

I learned the very basics of how Web API works, such as how to send requests and implement the response into my website.
